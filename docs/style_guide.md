# Руководство по стилю

## Локализация
- Только ARB-файлы в `lib/l10n/`: `app_en.arb` (шаблон и fallback), остальные — переводы (`app_ru.arb` и т.д.). Для en отдельного «переводного» файла нет — используется template.
- Workflow: Редактируешь ARB → `flutter gen-l10n` → код использует `AppLocalizations.of(context)!.key`
- Переменная в коде: всегда `final l10n = AppLocalizations.of(context)!;`
- Ключи: задаются в template (`app_en.arb`); в файлах переводов должны совпадать (при отсутствии ключа подставляется значение из template).
- Запрещено: использовать конкретный язык в `Text()` виджетах, всегда использовать локализованные ключи

## Темизация
- При изменении дизайна необходимо согласовывать изменения с `@lib/models/theme_preset.dart`
- Все цветовые схемы должны быть определены через `ThemePreset` модель
- Нельзя жестко задавать цвета в виджетах, использовать только тему

## Масштабирование текста
- При работе с текстовыми блоками учитывать `@lib/providers/text_scale_provider.dart`
- Нельзя устанавливать `fontSize` жестко, использовать относительные размеры или тему
- Все текстовые элементы должны корректно масштабироваться в соответствии с настройками пользователя

## Стиль UI

Ссылки: `@lib/models/theme_preset.dart`, `@lib/main.dart`, `@lib/app_constants.dart`.

- **Брейкпоинт:** 600 dp для «широкого» layout (Material 3). Один на всё приложение. Диалоги: max width 560.
- **Отступы:** 4 — мелкий (секции, чипы); 8 — между элементами в списке; 12 — заголовки карточек, поля ввода; 16 — padding экрана и карточки; 24 — широкий экран по горизонтали, крупные блоки. Padding карточки: 16. ContentPadding поля ввода: 12 вертикаль, 16 горизонталь (или 12/12 единообразно).
- **Карточки (Card):** padding 16; margin между карточками bottom: 8 (или zero внутри одной секции); borderRadius 12; elevation 0 по умолчанию, 4 для акцентной карточки.
- **Поля ввода:** contentPadding единый (12/12 или 16/12); borderRadius 12.
- **Типографика:** заголовок экрана — titleLarge; заголовок карточки — titleMedium; основной текст — bodyMedium; подписи/секции — labelSmall (uppercase, letterSpacing); кнопки — labelLarge. Не задавать fontSize — только стили темы.
- **Иконки:** стандарт 24; в AppBar и заголовках карточек 20; мелкие (кнопки, чипы) 18. Исключения: empty state 64/80, мелкий чип 14.
